Conteúdo Principal

Elementos básicos são o que constituem a base de todo documento HTML. Se vê esses elementos no código fonte de todas as páginas da web, logo em seguida da declaração de doctype, que está na primeira linha da página. O doctype específica qual a versão de HTML que a página está usando. Os elementos que formam o conteúdo da página são postos entre a tag de abertura.
